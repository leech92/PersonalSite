My Coding Journey So Far

Hello, my name is Christian and I am a software engineer. Before I found coding, I was a public tax accountant at KPMG. I knew I had a good job where I enjoyed working with my co-workers and I also knew that I was in a safe steady career path. However, after five years as a public accountant, I knew I wanted to do something else. I wanted to have a job where I could create, innovate, and find solutions that help people. I know this sounds like I am romanticizing software engineering, but when I first started coding and I was learning about arrays and for loops, something lit up inside me. I knew that this was what I wanted to do and that I was going to become the best software engineer I could be.

I recently graduated from App Academy and I have to say, this was one of the most challenging periods of my life. In four months I went from a complete beginner to someone who is capable of creating an AllTrails clone with user authentication, full CRUD reviews, and so much more. My project is called NotAllTrails and you can check it out in my projects section below. I am so proud of how far I've come but I am looking forward to learning so much mroe in the coming years.

Projects

NotAllTrails

A clone of AllTrails made for the purposes of learning. Users are able to search for parks and trails and navigate through the website through links directed to nearby trails. Signed in users are able to create, update, and delete reviews for trails. This clone of AllTrails was made to be as pixel perfect as possible.


Javascript, React, Redux, Ruby, Ruby on Rails, AWS, SQL, SCSS, Google Maps API

Pathfinder

A pathfinding visualizer that shows the user how a search algorithm finds the end point and returns the shortest path. Users are able to move both the start point and the end point. Users are also able to create wall tiles and weighted tiles. This pathfinding visualizer currently uses Dijkstra's Algorithm, but it will be updated to include more search algorithms.

Javascript, React, CSS, HTML
